# Health Intelligence Dashboard

헬스케어 산업 인텔리전스 대시보드 — **디바이스 헬스 · AI 네이티브 헬스 · 체중·피트니스 스타트업** 3개 카테고리의
트렌드, 데일리 기사 브리핑, 증권사·시장조사 리포트를 정량적으로 모니터링합니다.

> ⚠️ 표시되는 수치·기사는 실제 회사/이슈에 기반한 **현실적인 예시(sample) 데이터**이며 라이브 피드가 아닙니다.

## 주요 기능
- **왼쪽 블루 보드** — 빈 영역 클릭으로 접기/펼치기, 로고/스와치 클릭으로 보드 색상 변경
- **동적 인터랙션** — 스크롤로 화면에 들어오면 숫자가 0에서 카운트업, 차트가 0→100% 채워짐 (스크롤로 벗어났다 돌아오면 다시 0부터 재생)
- **정량 데이터 시각화** — KPI 게이지 바, 도넛(점유율), 가로 막대(밸류에이션·사용자), 추이 다이버징 바, 영역+라인(시장 규모)
- **데일리 기사 피드** — 날짜별 누적(최신 상단), 한국어 헤드라인 + 요약, 클릭 시 원문 이동
- **3개 카테고리 보드** — 기업별 밸류에이션·핵심지표(발표 기준일 `'25.10월 기준`)·추이, 행 호버 시 카테고리 색상 하이라이트
- **리서치 리포트** — 증권사·시장기관 리포트 정량 요약
- **Tweaks** — 다크모드(블룸버그 스타일), 카드 밀도, 사이드바·카테고리 색상

## 폰트
**Pretendard**(휴머니스트 산세리프, 한·영 + tabular nums)를 사용합니다.
실제 SamsungOne/Nanosans 폰트 파일이 있으면 `styles.css`의 `--f`, `--f-nano` 변수에 연결하세요.

## 파일 구조
```
index.html          진입점 (React + Babel CDN 로드)
styles.css          전체 스타일 (라이트/다크 테마 토큰)
data.js             샘플 데이터 (기업·기사·리포트·차트)
anim.jsx            스크롤 트리거 애니메이션 (카운트업·게이지·인뷰)
charts.jsx          SVG 차트 컴포넌트 (영역·막대·도넛)
components.jsx       사이드바·탑바·KPI
boards.jsx          카테고리 보드·기사 피드·차트·리포트
app.jsx             상태·테마·네비게이션·Tweaks
tweaks-panel.jsx    Tweaks 패널 셸
```

## 실행
정적 파일이므로 별도 빌드가 필요 없습니다. 로컬 서버에서 열면 됩니다:
```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```
